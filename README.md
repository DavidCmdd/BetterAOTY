<h4 align="center">A redesigned user interface for <a href="https://albumoftheyear.org" target="_blank">albumoftheyear.org</a>.</h4>

<p align="center">
 
![screenshot](https://github.com/DavidCmdd/BetterAOTY/blob/Images/fg.gif?raw=true)

## Key Features

✔ Auto Score Calculation (can be enabled/disabled)

✔ Modernized Layout (darker colors, rounded ui elements, better format)

✔ Accent Colors (green, yellow, blue, purple and red)

✔ Hide Album Scores (can be enabled/disabled)

✔ Better Rating Colors (can be enabled/disabled)

✔ Unrounded Scores 

✔ Spotify Stats (year, tracks, and length)

✔ Added site animations

✔ More improvements planned for upcoming releases!
